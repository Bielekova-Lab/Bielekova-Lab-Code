This folder contains a set of scripts used to generate results presented in the manuscript. 

# SYSTEM REQUIREMENTS
All scripts run on RStudio software Version 2023.12.1+402 (R version 4.3.3).
Libraries required to run each script are listed at the beginning of each script. 

# INSTALLATION GUIDE
There are specific installation requirements. 

# DEMO
Input folder provides all necessary files to run the scripts. 

# INSTRCUTIONS FOR USE
These scripts are not meant to be used with any other datasets - they are one-time scripts used to analyze our own proteomic dataset.
